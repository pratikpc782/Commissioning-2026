# Day 01 Reflection

**## What did you learn today?**

Today was the first day of my GraySentinel 45-Day Commissioning Program, and I learned several important cybersecurity concepts.
I gained a better understanding of the role of a Security Operations Center (SOC) and how it continuously monitors and protects
an organization's systems from cyber threats. I also learned about SIEM, which collects and analyzes logs from different devices 
to detect suspicious activities and generate alerts. Another important topic was the difference between Blue Team and Red Team. 
I understood that the Blue Team focuses on defending systems, while the Red Team simulates attacks to identify security weaknesses. 
In addition, I learned why virtual machines are essential in cybersecurity for creating safe lab environments, testing tools, 
and analyzing malware without affecting the host system. I also improved my GitHub skills by creating a repository, organizing folders,
and documenting my work using Markdown.

## What challenges did you face?

The biggest challenge I faced today was learning how to use GitHub correctly. Initially, I had trouble creating folders and files because 
I entered the file path incorrectly. As a result, the "Commit changes" button remained disabled. After understanding how GitHub organizes
files and folders, I corrected the file path and successfully committed my changes. I also spent some time understanding Markdown formatting 
for headings, bold text, and document organization.

## Which concepts were completely new?

The concepts that were completely new to me were SIEM log correlation, the different SOC analyst levels (Tier 1, Tier 2, and Tier 3), and the
importance of virtual machine snapshots. I also learned how security professionals document their learning using GitHub, which is a valuable skill
for building a professional cybersecurity portfolio.

## How can you improve tomorrow?

Tomorrow, I will spend more time practicing the concepts I learned today instead of only reading about them. I plan to explore GitHub features further, 
improve my Markdown documentation, and begin hands-on cybersecurity lab exercises. My goal is to stay consistent throughout the 45-day program, strengthen 
my practical skills, and build a professional portfolio that reflects my learning journey.
